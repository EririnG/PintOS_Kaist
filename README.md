# PintOS_Project
## 크래프톤 정글 4기 pintOS

카이스트 CS330의 핀토스 과제입니다.

핀토스-카이스트(이후 핀토스)는 x86-64아키텍쳐를 위한 간단한 OS 프레임워크입니다. 핀토스는 커널 쓰레드, 유저 프로그램 로딩 및 실행, 그리고 파일 시스템을 매우 단순한 형태로 구현합니다.

이론적으로, 핀토스는 x86-64 CPU에서 돌아갈 것입니다.
